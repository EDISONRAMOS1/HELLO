FURTHER DISCUSSION & EXAMPLES: GRAPH OF A LINEAR FUNCTION

A linear function is defined by f(x) = mx + b, where m and b are real numbers. Its graph is a line with slope (m) and y-intercept (b).
If m ≠ 0, then the degree of the function is 1. If m = 0 and b ≠ 0, then the degree of the function is 0. If m = 0 and b = 0, then the degree of the function is not defined.

❑ If m ≠ 0, then the degree of the function is 1. The line is diagonal.
❑ If m = 0 and b ≠ 0, then the degree of the function is 0. The line is horizontal.
❑ If m = 0 and b = 0, then the degree of the function is not defined. There is no line.

While all linear equations produce straight lines when graphed, NOT all linear equations produce linear functions.
To be a linear function, a graph must be both linear (a straight line) and a function (matching each x-value to only one y-value).
You can use the slope or linear function table (table of values) formula.
