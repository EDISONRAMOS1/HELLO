FURTHER DISCUSSION & EXAMPLES: DEFINITION OF RELATION AND FUNCTION

RELATION
As we go about our daily lives, many data items or quantities are paired to our names. Our social security number, student ID number, email address, phone number, and birthday are matched to our name. There is a relationship between our name and each of those items. For example, when your professor gets her class roster, the students' names in the class are listed in one column and then the student ID number is likely to be in the next column. If we think of the correspondence as a set of ordered pairs, where the first element is a student name and the second element is that studentâ€™s ID number, we call this a relation.

FUNCTION
A special type of relation, called a function, occurs extensively in mathematics. A function is a relation that assigns to each element in its domain exactly one element in the range. For each ordered pair in the relation, each x-value is matched with only one y-value. Think of it this way, every person has a birthday but no one has two birthdays. It is okay for two people to share a birthday.
